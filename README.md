# 🧠 ObjectVisionAI

Projeto de Inteligência Artificial utilizando visão computacional para reconhecimento de objetos através de imagens.

## 🚀 Tecnologias

- Python
- TensorFlow
- Keras
- NumPy
- Pillow
- Teachable Machine

## 🎯 Objetos reconhecidos

- 🎮 Controle de videogame
- 📱 Celular
- ⌨ Teclado de computador
- 🖱 Mouse de computador
- 🎧 Headset

## ⚙ Como funciona

A IA analisa uma imagem chamada:

```text
foto.jpg
```
A imagem deve estar na mesma pasta do projeto e obrigatoriamente nomeada como `foto.jpg`.

## 📦 Instalação

```bash
git clone https://github.com/RayanCarvalhoDev/ObjectVisionAI.git

cd ObjectVisionAI

pip install -r requirements.txt
```

## ▶ Executar

```bash
python analisar.py
```

### Alternativa (Iniciar com um clique (recomendado))

Crie um arquivo chamado:

```text
Iniciar.bat
```

E coloque:

```bat
@echo off
python analisar.py
pause
```

Depois basta dar duplo clique no arquivo para executar o projeto.
